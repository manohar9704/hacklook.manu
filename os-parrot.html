Parrot OS

A privacy and security focused distribution. Great to experiment with in VMs.

How to download Parrot (basic)

  1. Visit the official Parrot OS site: https://www.parrotsec.org/
  2. Choose the edition (Home or Security) and download the ISO.
  3. Verify the checksum if provided on the downloads page.
  4. Use Etcher or Rufus to create a bootable USB, or use a VM to test first.

Download Parrot cheat-sheet (PDF)

sudo apt install <package>
